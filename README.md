# HEMOCARES API

Merupakan backend dari aplikasi HEMOCARES.

Cara Installasi Awal :
*   NPM i atau NPM install

Kebutuhan Plugin dari Sistem :
*   axios
*   bcrypt
*   bluebird
*   compression
*   cors
*   date-fns
*   date-fns-tz
*   dotenv
*   express
*   formidable
*   fs-extra
*   helmet
*   joi
*   jsonwebtoken
*   mongoose
*   morgan
*   util
*   uuid

## DETAILS

Aplikasi dibangun dengan menggunakan node js v.16.13.0, dan database MongoDB
